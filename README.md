# ShoppingProject

**Java 기반 쇼핑 애플리케이션**입니다.  
Java + Servlet/JSP 기반으로 웹 UI와 백엔드를 구성해, 사용자가 상품을 보고 장바구니에 담아 주문할 수 있는 기능을 제공합니다.

---

##  기능
- 사용자: 회원가입, 로그인/로그아웃
- 상품 목록 보기, 상품 상세 페이지
- 장바구니에 상품 담기 / 수량 변경 /장바구니 삭제
- 주문: 장바구니 → 주문, 주문 내역 보기
- 관리자: 상품 등록/수정/삭제, 주문 상태 관리

---

##  요구사항 & 개발 환경
- Java JDK 8 이상
- Apache Tomcat (v8.5 이상)
- Eclipse IDE 사용
- Maven
- 데이터베이스(MySQL)
